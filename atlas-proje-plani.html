Mimari Sütunlar

Atlas projesi, CTO referans planı olarak 4 temel sütun üzerine inşa edilmektedir. Her sütun birbirini besleyerek 2026 ve sonrasına ölçeklenebilir bir sistem oluşturur.

01

UI/UX & Tasarım Sistemi — Ajan Destekli Akışlar

Figma MCP + Storybook + Cursor IDE entegrasyonuyla Zero Drift (Sıfır Sapma) felsefesi. Tasarım tokenları ve bileşen kuralları otomatik olarak koda aktarılır. Dark Pattern'lar reddedilir; Ambient UX ve niyet odaklı (intent-driven) arayüz dili benimsenir.

Figma MCP Storybook Zero Drift Ambient UX Intent-Driven UI Zod Schema
02

Generative UI & Zeka Katmanı

CopilotKit + AG-UI + MCP entegrasyonuyla ajanların dinamik Dashboard'lar ürettiği Generative UI mimarisi. A2UI standartlarında önceden onaylanmış React bileşen kataloğu. Human-in-the-Loop güvenlik denetimi ve çoklu ajan orkestrasyonu.

CopilotKit AG-UI MCP A2UI Generative UI Multi-Agent
03

Çekirdek Uygulama & Durum Yönetimi

Next.js App Router üzerinde React 19.2+ ve React Compiler ile otomatik memoization. useActionState, useOptimistic hooks ile asenkron süreçler. Turbopack derleyici + Edge Computing ile sıfır gecikmeli küresel mimari.

React 19.2+ Next.js App Router Turbopack tRPC TypeScript Edge Computing
04

Mekansal Render Motoru & Veri İndeksleme

WebGPU Compute Shaders üzerinde LTTB downsampling ile milyonlarca veri noktasının 60 FPS'de render edilmesi. Deck.gl v9 + MapLibre v5 Globe View entegrasyonu. Geometrik bozulmaları sıfıra indiren A5 Beşgen DGGS sistemi ile 64-bit BigInt veri akışı.

WebGPU Deck.gl v9 MapLibre v5 A5 DGGS LTTB Algorithm Compute Shaders

WebGPU & Mekansal Veri Görselleştirme

WebGL'in sınırlarını aşarak GPU'nun tam gücünü açığa çıkaran Compute Shaders mimarisi. Trilyonlarca satır coğrafi verinin 60 FPS'de render edilmesi.

🔺 LTTB Downsampling — GPU'da

1 milyonluk veri noktasının haritada render edilmesi için Largest Triangle Three Buckets (LTTB) algoritması Compute Shaders üzerinde çalıştırılır. CPU bottleneck tamamen ortadan kalkar.

// Compute Shader — LTTB Pipeline @group(0) @binding(0) var<storage> input_data: array<f32>; @group(0) @binding(1) var<storage, read_write> output: array<f32>; @compute @workgroup_size(64) fn main(@builtin(global_invocation_id) id: vec3<u32>) { // LTTB bucket calculation on GPU }

🌍 Gerçek Zamanlı Simülasyonlar

Hava durumu, basınç haritaları veya demografik yoğunluk animasyonları için düzensiz veri kümeleri GPU'da Marching Squares + Chaikin Subdivision ile işlenir.

// Rodrigues Rotation — Sphere Surface fn rodrigues_rotate(v: vec3<f32>, k: vec3<f32>, theta: f32) -> vec3<f32> { return v * cos(theta) + cross(k, v) * sin(theta) + k * dot(k, v) * (1.0 - cos(theta)); }

🗺️ Deck.gl v9 Mimarisi

Uniform Buffers mimarisine geçiş ile WebGL1 global değişkenlerinden tam kopuş. MapLibre v5 ile Globe View kamera senkronizasyonu ve MapboxOverlay entegrasyonu.

import { Deck } from '@deck.gl/core'; import { A5Layer } from '@deck.gl/layers'; const deck = new Deck({ device: { type: 'webgpu' }, // WebGPU modu layers: [ new A5Layer({ data: pentagonalGrid, getPentagon: d => d.cell_id_bigint, getFillColor: d => getColorByValue(d.value), getElevation: d => d.elevation, // GPU'da 3D extrusion }) ] });

⬠ A5 DGGS — Beşgen Grid Sistemi

Tüm dünya yüzeyini kesinlikle eşit alanlı beşgenlere bölen yeni nesil indeksleme sistemi. 64-bit BigInt kimliklerle veri akışı; API istek sayısı minimumda tutulur.

// A5 Cell ID — 64-bit BigInt const cellId: bigint = 0x1A2B3C4D5E6F7890n; // getPentagon accessor — GPU'da koordinat hesaplama const layer = new A5Layer({ getPentagon: (d: A5Data) => BigInt(d.cell_id), // Koordinat dönüşümü GPU shader'da yapılır // CPU'ya yük binmez });

Yapay Zeka & Generative UI Katmanı

Modern Ajanik Yığın (Modern Agentic Stack) üç katmanlı protokol mimarisi üzerine inşa edilir. Ajanlar sıfırdan kod yazmak yerine önceden onaylı bileşen kataloğundan seçim yapar.

📋

Katman 1 — A2UI / MCP Apps

Sistemin kalbi. Ajanların eriştiği JSON tabanlı bildirimsel (declarative) bileşen kataloğu. Ham HTML/CSS üretimi güvenlik riskleri nedeniyle yasaklı; yalnızca onaylı bileşen ID'leri çağrılabilir.

Google A2UI Standardı
⚡

Katman 2 — AG-UI Runtime

Yapay zeka ajanı ile frontend arasındaki gerçek zamanlı veri akışını yönetir. CopilotKit üzerinden State senkronizasyonu; LLM çıktısının anlık görsel bileşenlere dönüşmesi garanti edilir.

CopilotKit + AG-UI
🔌

Katman 3 — MCP Tool Layer

Ajanların dış dünyayla iletişim kurmasını sağlayan Model Context Protocol katmanı. Veritabanları, hesaplama algoritmaları ve kullanıcı oturum bilgilerine güvenli erişim noktaları.

🎯 Hedef-Öncelikli Katılım (Goal-First Onboarding)

Yeni kullanıcıya özellik turu yerine doğrudan "neye ulaşmak istiyorsunuz?" sorusu sorulur. Sistem arayüzünü kullanıcının hedefine göre dinamik olarak yeniden yapılandırır.

🛡️ İnsan Döngüsü Denetimleri (Human-in-the-Loop)

Ajan kararları ve uyguladığı değişiklikler hızlı geri alma (undo) mekanizmaları ve ortak otonomi kontrolleriyle kullanıcının müdahalesine açık tutulur.

🚫 Dark Pattern Reddi

"Yalnızca 2 koltuk kaldı" gibi sahte aciliyet bildirimleri, önceden işaretli onay kutuları ve duygusal CTA metinleri 2026 UX etiği çerçevesinde tamamen reddedilir.

🤖 Savunma Ajanları (Gatekeeper Agents)

Ana analiz ajanının aldığı kararları denetleyen ve kullanıcının belirlediği kurallara uygunluğu kontrol eden güvenlik ajan katmanı. Çoklu ajan orkestrasyonu.

Risk Analizi & Azaltma Stratejileri

Kritik bağımlılıklar ve teknik riskler için proaktif azaltma planları.

Yüksek Risk

WebGPU Tarayıcı Desteği

WebGPU tüm tarayıcılarda henüz tam olarak desteklenmemektedir. Safari ve eski Chrome sürümlerinde render motoru çalışmayabilir.

Azaltma: Deck.gl v9'un WebGL2 fallback mekanizması aktif edilir. Cihaz kapasitesine göre otomatik render modu seçimi uygulanır. Progressive Enhancement stratejisi benimsenir.
Yüksek Risk

A5 DGGS Ekosistem Olgunluğu

A5 sistemi H3/S2'ye kıyasla daha yeni bir ekosistemdir. Üçüncü taraf kütüphane desteği sınırlı olabilir.

Azaltma: A5Geo resmi kütüphanesi ile başlanır. Gerekirse H3 → A5 dönüşüm katmanı inşa edilir. H3 sistemi yedek olarak hazır tutulur.
Yüksek Risk

Ajan Güvenlik Açıkları (NeighborJack)

Ajanların ürettiği UI bileşenleri siber güvenlik saldırılarına (araç zehirlenmesi, NeighborJack) maruz kalabilir.

Azaltma: A2UI standardında yalnızca önceden onaylı bileşen ID'leri çağrılabilir. Ham HTML/CSS üretimi tamamen yasaklanır. Gatekeeper Agent her ajan kararını denetler.
Orta Risk

React 19 Compiler Uyumsuzluğu

React Compiler bazı mevcut hook kullanım kalıplarıyla çakışabilir. Üçüncü taraf kütüphaneler uyumsuz olabilir.

Azaltma: Kütüphane uyumluluk listesi önceden denetlenir. Sorunlu bileşenler için 'use no memo' direktifi kullanılır. Aşamalı geçiş planı hazırlanır.
Orta Risk

MCP Ekosistem Değişimleri

MCP protokolü hızla gelişmektedir. Versiyon değişikliklerinin mevcut ajan entegrasyonlarını bozma riski mevcuttur.

Azaltma: MCP adaptör katmanı soyutlanır (abstraction layer). Sürüm kilitleme (version pinning) uygulanır. Haftalık ekosistem güncellemeleri takip edilir.
Düşük Risk

Büyük Veri Kümesi Performansı

Trilyonlarca satır coğrafi verinin istemci tarafında işlenmesi beklenmedik bellek sorunlarına yol açabilir.

Azaltma: A5 hiyerarşik indeksleme ile API yükü minimize edilir. LTTB Compute Shader ile downsampling GPU'ya devredilir. Wasm modülleri ağır hesaplamalar için kullanılır.
